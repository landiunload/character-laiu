# Charlaiu — анкета персонажа «Где он лёг»

Приложение для писателей: подробные карточки героев — внешность, характер,
мечты и страхи, связи, роль в сюжете и пять вопросов на глубину.

Работает целиком в браузере: без сервера, без регистрации, без отправки данных.
Написано на C# и работает через **Blazor WebAssembly**; первая версия проекта
была на чистом JavaScript и сохранена в истории репозитория.

## Возможности

- Полная анкета по структуре «Где он лёг»: 11 разделов, включая таблицу
  «Любит / Не любит» и авторские подсказки.
- Сколько угодно героев: создание, дублирование, удаление, список в боковой панели.
- Гибкая структура: любое поле и любой раздел можно добавить, переименовать или
  убрать — приложение не привязано к конкретной схеме.
- Автосохранение в localStorage браузера при каждом изменении.
- Экспорт анкеты в Markdown (готовый документ) и JSON (резервная копия),
  импорт JSON обратно.
- Светлая и тёмная темы, адаптивная вёрстка для телефона.

## Стек технологий

- **.NET 9 / Blazor WebAssembly** — весь код приложения на C#, выполняется в браузере
- **System.Text.Json** — сериализация для хранения и экспорта
- **xUnit** — модульные тесты домена и экспортёров

## Архитектура

```
src/Charlaiu.Web/
├── Domain/        # Модель анкеты: герой, разделы, поля, фабрика схемы по умолчанию
├── Services/      # Хранилище (localStorage), экспортёры Markdown и JSON, скачивание файлов
├── Pages/         # Страница-редактор
└── wwwroot/       # Стили, JavaScript-обёртка браузерных API
tests/Charlaiu.UnitTests/
```

Ключевые решения:

- **Хранилище за интерфейсом** `ICharacterProfileRepository`: реализация на
  localStorage заменяется на файловую или серверную без изменения компонентов.
- **Экспорт через общий контракт** `IQuestionnaireExporter`: новый формат — это
  новая реализация интерфейса, существующий код не меняется.
- **Схема анкеты — данные, а не код**: интерфейс рендерит любую структуру
  разделов и полей, поэтому пользователь свободно редактирует анкету под себя.
- Домен не зависит от Blazor и браузера, поэтому тестируется обычным xUnit.

## Запуск

Нужен установленный [.NET SDK 9](https://dotnet.microsoft.com/download/dotnet/9.0):

```bash
dotnet run --project src/Charlaiu.Web
```

Адрес будет в выводе команды (обычно http://localhost:5000).

## Тесты

```bash
dotnet test
```
